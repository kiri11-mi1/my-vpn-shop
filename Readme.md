# Finance tracker

## Launch app
1. Install `docker` and `docker-compose`
2. Create file `.env` similar `.env.example`
3. Run this bot
    ```
   make up
   ```

## Run tests
1. Run app
    ```
   make up
   ```
2. Test app
    ```
   make test
   ```

## Stop app
If bot is started, then:
```
    make down
```
